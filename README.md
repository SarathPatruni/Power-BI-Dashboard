# HR Attrition Analytics Dashboard (Power BI)

This is my first Power BI project focused on analyzing employee attrition and key workforce metrics using HR data.

## Objective
To understand employee attrition patterns and analyze workforce trends based on overtime, salary, department, and other HR attributes.

## Key Insights Covered
- Attrition Count (Yes vs No)
- Overtime Analysis
- Employee Count Month-wise Trend
- Salary Distribution
- Workforce breakdown using dynamic slicers

## Features
- Interactive slicers for:
  - Gender
  - Department
  - State
  - Marital Status
  - Business Travel
  - Salary Range
  - Job Role
- Dynamic filtering across all visuals
- Clean and structured dashboard layout

## Tools & Skills Used
- Power BI
- DAX
- Power Query
- Data Cleaning
- Data Visualization
- Basic HR Analytics

## Files in This Repository
- `HR_Attrition_Dashboard.pbix` – Main interactive Power BI file  
- `HR_Attrition_Dashboard_Report.pdf` – Exported dashboard report  
- `/screenshots` – Dashboard preview images  

## How to View the Dashboard
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Use the slicers to interact with the dashboard
